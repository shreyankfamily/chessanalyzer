⚙️ ChessAnalyzer Bridge Settings

Example: http://192.168.1.100:3000
Leave empty to use localhost (same machine)

How to use:

  1. Same laptop: Leave server address empty (uses localhost:3000)
  2. Different laptop: Enter the server IP (e.g., http://192.168.1.100:3000)
  3. Find your server IP by running: node launcher.js on the server laptop
  4. Click "Test Connection" to verify it works
